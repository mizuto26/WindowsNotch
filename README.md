# WindowsNotch

Windows で mac のノッチ風 UI を楽しむためのデスクトップアプリです。  
画面上部のノッチから、ファイルの一時保管、iCloud Drive への共有、再生中メディアの表示ができます。

## できること

- 画面上部にノッチ風の UI を表示
- ファイルを一時保管する `Shelf` 機能
- iCloud Drive 用エリアにドロップして iPhone へ共有
- 再生中の音楽情報をノッチ内に表示
- 設定画面から起動時の自動起動などを変更

## 使い方

1. GitHub の Releases から `WindowsNotch-win-x64.zip` をダウンロードします。
2. zip を展開します。
3. 展開したフォルダの `WindowsNotch.App.exe` を実行します。
4. 画面上部中央にカーソルを持っていくとノッチが開きます。

## ファイル共有

1. Windows に `iCloud for Windows` をインストールします。
2. `iCloud Drive` を有効にします。
3. Windows と iPhone の両方で同じ Apple Account にサインインします。
4. ノッチ左側の `iCloud Drive` エリアにファイルをドロップします。
5. iPhone の `ファイル` アプリで `iCloud Drive > WindowsNotch` を開きます。


## Shelf 機能

- ノッチ右側のエリアにファイルをドロップすると一時保管されます。
- 保管したファイルは左右キーで選択できます。
- `Backspace` キーで削除できます。
- 保管したファイルは外部フォルダへドラッグして取り出せます。

## 音楽表示

- 再生中のメディア情報をノッチ内に表示できます。
- アプリによってはアルバムアートや操作に対応しない場合があります。
- Windows のメディアセッションを公開しているアプリで動作しやすいです。

## 対応環境

- Windows 10 / 11 x64
